# GameHub JavaScript 1 Resit

This project is an interactive online store built for the JavaScript 1 Course Assignment Resit.

## Features

- View products from the GameHub API
- Filter products by genre
- View single product details
- Add products to basket
- Remove products from basket
- View checkout summary
- Complete order and view confirmation page

## Built with

- HTML
- CSS
- JavaScript
- Noroff GameHub API
